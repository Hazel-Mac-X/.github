## Overview

<a href="#">Hazel</a> is a powerful macOS automation utility developed by Noodlesoft that monitors designated folders and automatically processes files according to user-defined rules. Rather than manually sorting downloads, organizing project files, or cleaning up desktop clutter, Hazel performs these repetitive file management tasks silently in the background every time new content appears in a watched folder.

The <a href="#">rule-based engine</a> evaluates each file against a set of conditions you configure — file name patterns, extensions, creation dates, file size thresholds, content keywords, color labels, and dozens more attributes. When a file matches the conditions, Hazel executes the assigned actions: moving to a target folder, renaming according to a template, adding tags, running a shell script, or even sending the file to the Trash. <a href="#">Condition combinations</a> support AND and OR logic, enabling sophisticated multi-attribute filtering that catches exactly the right files while leaving others untouched.

<a href="#">Folder watching</a> operates continuously without any user interaction. Hazel processes files the moment they land in a watched directory, which means Downloads, Desktop, and any project folders stay organized automatically throughout the workday. The <a href="#">priority system</a> determines the order rules are evaluated, letting you chain complex sorting workflows where the output of one rule feeds into the next. This chained logic handles multi-stage organization tasks that would otherwise require manual attention.

<a href="#">Renaming patterns</a> combine static text with dynamic tokens drawn from file metadata — original filename, date modified, EXIF data from photos, document text, and more. Screenshots automatically receive descriptive names. Bank statements get organized by date and institution. Invoices move to client folders based on filename patterns. The renaming system transforms inconsistently named files into a coherent naming scheme without touching each file individually.

## Key Features

- <a href="#">Automatic folder monitoring</a> watches any number of directories simultaneously and processes files the instant they appear
- <a href="#">Rich condition library</a> matches files by name extension size date tags content spotlight attributes and dozens more criteria
- <a href="#">Multi-action workflows</a> chain move rename tag compress run scripts and send to apps in a single rule sequence
- <a href="#">Pattern-based renaming</a> builds consistent filenames from metadata tokens dates and extracted document content
- <a href="#">AppleScript and shell script</a> execution extends automation beyond built-in actions to custom processing logic
- <a href="#">Trash management rules</a> automatically empty Trash contents older than a configured interval keeping storage clear
- <a href="#">Hazel Sync</a> shares rule configurations across multiple Macs through iCloud keeping every machine consistently configured
- <a href="#">Import and export of rules</a> packages rule sets for backup sharing and deploying identical setups on new machines
- <a href="#">Preview mode</a> shows exactly which files a rule would affect before enabling it preventing accidental misorganization

## Additional Information

Hazel represents the kind of focused macOS utility that solves a genuine problem elegantly. Noodlesoft has refined the application over many years, adding capabilities that reflect how power users actually work without overwhelming casual users with unnecessary complexity. The preference pane interface integrates naturally into macOS System Settings and feels at home alongside other system tools.

Updates consistently add support for new macOS features and file attributes as Apple releases them. The application runs efficiently as a background process without impacting system performance. For anyone managing significant numbers of files on a Mac, Hazel provides automation that genuinely reclaims time previously spent on manual organization.
